プライバシーポリシー

対象アプリ: GCAL連携（非公式）

1. データの取得方法

GCAL連携（非公式）は、Google API を通じてカレンダーデータを取得します。 具体的には、ユーザーのGoogleカレンダーの読み取り（予定の取得）および 必要に応じた書き込み（予定の同期）を行います。

GCAL Sync (Unofficial) retrieves calendar data via the Google API, including read access to events and write access for syncing when necessary.

2. データの利用目的

予定の同期と表示のみに使用し、外部には共有しません

Data is used only for syncing and display and is not shared externally.

3. データの共有・保存

  • 第三者への提供は行いません。
  • 保存期間は60日です。
  • 不要になったデータは適切に削除します。
  • No data is provided to third parties.
  • Retention period is 60 days.
  • Data no longer needed is deleted appropriately.

4. Google限定使用の宣言

Google APIから受け取った情報の使用は、Google API Services User Data Policy （限定使用）に従います。

Use of information received from Google APIs complies with the Google API Services User Data Policy (Limited Use).
